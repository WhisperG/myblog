Golang使用之图形界面开发

2017-08-18 | 浏览

学习golang有段时间了，这次由于工作需要，需要写个UI程序，这里使用了go-sciter，Sciter是一个嵌入式的HTML/CSS/脚本引擎，旨在为桌面应用创建一个 UI 框架层。说简单点就是我们通过它可以像写html，css那样写桌面，成熟案例如Norton Antivirus和Internet Security，Comodo Internet Security，ESET Antivirus，BitDefender Antivirus等

一、环境准备环境准备

第一步：下载 sciter-sdk

解压，找到sciter-sdk\bin\64\sciter.dll复制到c:\windows\system32
注意上面的请根据你自已的系统选择相应文件

第二步：由于使用到cgo，所以window下需要安装mingw或tdm-gcc(建议安装tdm-gcc)

https://sourceforge.net/projects/mingw/files/
http://tdm-gcc.tdragon.net/download

下载：mingw-get-setup.exe或tdm64-gcc-5.1.0-2.exe
安装：然后把mingw\bin或tdm-gcc\bin加入到环境变量中

第三步：cmd进入gopath目录并运行

go get -x github.com/sciter-sdk/go-sciter
