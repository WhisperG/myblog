Ngrok服务搭建

2017-08-10 | 浏览

​ Ngrok 是一个反向代理，通过在公共的端点和本地运行的 Web 服务器之间建立一个安全的通道。Ngrok 可捕获和分析所有通道上的流量，便于后期分析和重放。

一、准备

​ 服务器使用的是Ubuntu Server 16.04 amd64

​ Ngrok源码：https://github.com/inconshreveable/ngrok.git

​ golang：https://golang.org/dl/
