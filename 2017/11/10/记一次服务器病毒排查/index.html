记一次服务器病毒排查

2017-11-10 | 浏览

排毒：

（1）关闭访问挖矿服务器的访问 iptables -A INPUT -s xmr.crypto-pool.fr -j DROP 和iptables -A OUTPUT -d xmr.crypto-pool.fr -j DROP
（2）删除/usr/local/etc 下root文件中的内容
（3）删除yam 文件（我的yam文件不是在上面说的/opt目录下的，使用find命令查找，然后删除）
（4）删除 /root/.ssh/KHK75NEOiq
（5）删除/opt/minerd 和 /opt/KHK75NEOiq33
（6）杀死minerd进程，pkill minerd 或者kill -9 进程Id

修复redis的后门

a.配置bind选项, 限定可以连接Redis服务器的IP, 并修改redis的默认端口6379
b.配置AUTH, 设置密码, 密码会以明文方式保存在redis配置文件中.
c.配置rename-command CONFIG “RENAME_CONFIG”, 这样即使存在未授权访问, 也能够给攻击者使用config指令加大难度
d.好消息是Redis作者表示将会开发”real user”，区分普通用户和admin权限，普通用户将会被禁止运行某些命令，如conf
打开 ~/.ssh/authorized_keys, 删除你不认识的密钥
删除用户列表中陌生的帐号
